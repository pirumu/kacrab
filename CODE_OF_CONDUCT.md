# Code of Conduct

## Standard

This project expects professional, respectful collaboration.

Be direct about technical trade-offs, but keep feedback about the work, not the
person. Assume contributors are here to improve the project. Disagreement is
welcome when it is specific, evidence-based, and actionable.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or conduct.
- Repeatedly derailing technical discussions.
- Publishing private information without permission.
- Any behavior that would make reasonable contributors feel unsafe or unwelcome.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block
participants who violate this code.

If you need to report a problem, open a private security advisory if the matter
is sensitive, or contact the maintainer through the repository owner profile.
Include links, screenshots, or context when possible.
